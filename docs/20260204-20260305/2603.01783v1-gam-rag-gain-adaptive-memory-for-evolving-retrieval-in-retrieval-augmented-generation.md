## 摘要
检索增强生成 (RAG) 通过外部证据为大语言模型提供支撑，但许多实现依赖于构建后保持静态的预建索引。因此，相关查询会重复类似的多跳遍历，增加了延迟和计算开销。受认知神经科学中基于图式学习 (schema-based learning) 的启发，我们提出了 GAM-RAG，这是一个无需训练的框架，它能从重复或相关的查询中积累检索经验，并随时间更新检索记忆。GAM-RAG 构建了一个轻量级、无关系的层次化索引，其链接捕捉的是潜在的共现关系而非固定的语义关系。在推理过程中，成功的检索片段提供句子级反馈，更新句子记忆，使得对类似推理类型有用的证据在以后更容易被激活。为了在噪声反馈下平衡稳定性和适应性，我们引入了一种受卡尔曼滤波启发的、具有不确定性感知能力的增益规则，该规则共同更新记忆状态和基于困惑度 (perplexity) 的不确定性估计。它对可靠的新信号应用快速更新，对稳定或有噪声的记忆进行保守细化。我们提供了更新动力学的理论分析，并从实验上证明，GAM-RAG 在平均性能上比最强基线提高了 3.95%，在 5 轮记忆下提高了 8.19%，同时将推理成本降低了 61%。我们的代码和数据集可在以下地址获取：https://anonymous.4open.science/r/GAM_RAG-2EF6。

## Abstract
Retrieval-Augmented Generation (RAG) grounds large language models with external evidence, but many implementations rely on pre-built indices that remain static after construction. Related queries therefore repeat similar multi-hop traversal, increasing latency and compute. Motivated by schema-based learning in cognitive neuroscience, we propose GAM-RAG, a training-free framework that accumulates retrieval experience from recurring or related queries and updates retrieval memory over time. GAM-RAG builds a lightweight, relation-free hierarchical index whose links capture potential co-occurrence rather than fixed semantic relations. During inference, successful retrieval episodes provide sentence-level feedback, updating sentence memories so evidence useful for similar reasoning types becomes easier to activate later. To balance stability and adaptability under noisy feedback, we introduce an uncertainty-aware, Kalman-inspired gain rule that jointly updates memory states and perplexity-based uncertainty estimates. It applies fast updates for reliable novel signals and conservative refinement for stable or noisy memories. We provide a theoretical analysis of the update dynamics, and empirically show that GAM-RAG improves average performance by 3.95% over the strongest baseline and by 8.19% with 5-turn memory, while reducing inference cost by 61%. Our code and datasets are available at: https://anonymous.4open.science/r/GAM_RAG-2EF6.

传统的检索增强生成（RAG）系统通常依赖于构建后即保持静态的索引结构。这意味着系统是“无状态”的：即使面对重复或高度相关的查询，系统也会机械地重复相同的多跳遍历和推理过程，导致计算冗余和高延迟。此外，静态索引难以适应不断变化的查询需求或纠正初始构建时的错误。
**研究动机：** 受认知神经科学中“图式学习”和“赫布理论”（细胞共同激活则连结加强）的启发，作者旨在开发一种具有**结构可塑性**的 RAG 框架，使其能够从推理经验中学习，通过动态更新记忆来优化未来的检索效率和准确性。

#### 2. 方法论：核心思想与关键技术
GAM-RAG 是一个**免训练（Training-free）**的框架，其核心思想是将检索过程视为一个动态演进的记忆更新过程。

*   **轻量级层级索引构建：** 不同于复杂的知识图谱（KG），它通过实体共现构建“实体-句子-段落”三层图结构，降低了索引构建成本。
*   **双维度句子记忆：** 为每个句子节点维护两个记忆向量：**任务导向记忆（Task-oriented）**和**时间导向记忆（Time-oriented）**，分别捕捉语义贡献和时间约束。
*   **记忆引导的迭代检索：**
    1.  **初始激活：** 基于查询实体激活图节点。
    2.  **语义传播：** 利用记忆向量作为先验权重，重新加权实体到句子的传播得分。
    3.  **证据聚合：** 结合句子权重和段落相似度进行排名。
*   **卡尔曼启发式增益更新（核心算法）：**
    *   **残差计算：** 计算 LLM 反馈信号与当前记忆预测之间的差异。
    *   **卡尔曼增益：** 引入“不确定性（困惑度）”概念。当记忆不确定时，增益大（快速学习）；当记忆趋于稳定或反馈噪声大时，增益小（保守更新）。
    *   **状态与不确定性同步更新：** 动态调整记忆向量的方向，并根据反馈降低或维持不确定性估计。

*   **动态自适应：** 突破了 RAG 索引静态化的瓶颈，实现了“越用越聪明”。
*   **卡尔曼增益机制：** 巧妙地利用不确定性估计来平衡稳定性和灵活性，有效对抗了 LLM 反馈中的噪声。
*   **高效轻量：** 无需昂贵的显式关系抽取，且无需离线微调，易于在资源受限环境下部署。
*   **多维度建模：** 同时考虑任务语义和时间维度，适应性更广。

#### 8. 不足与局限
*   **反馈依赖：** 系统的更新质量高度依赖于“LLM 作为裁判”的反馈准确性，若 LLM 产生幻觉提供错误反馈，可能导致记忆漂移。
*   **冷启动问题：** 在没有任何历史经验的初始阶段，其性能优势并不明显，需要一定的“热身”过程。
*   **存储开销：** 随着处理的句子增多，维护每个句子的动态记忆向量和不确定性参数会带来额外的存储负担。
*   **应用限制：** 主要针对具有重复性或相关性查询的长程任务，对于完全随机、无关联的单次查询，其优势无法体现。
