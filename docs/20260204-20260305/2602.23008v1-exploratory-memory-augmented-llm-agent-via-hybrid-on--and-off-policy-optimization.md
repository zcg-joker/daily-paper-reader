## 摘要
探索仍然是使用强化学习训练的大语言模型智能体的关键瓶颈。虽然先前的方法利用了预训练知识，但它们在需要发现新状态的环境中表现不佳。我们提出了探索性记忆增强同策略与异策略优化（EMPO$^2$），这是一种混合强化学习框架，它利用记忆进行探索，并结合了同策略和异策略更新，使大语言模型在有记忆的情况下表现良好，同时确保在没有记忆的情况下的鲁棒性。在 ScienceWorld 和 WebShop 上，EMPO$^2$ 分别比 GRPO 提高了 128.6% 和 11.3%。此外，在分布外测试中，EMPO$^2$ 展示了对新任务的卓越适应能力，仅需少量记忆尝试且无需参数更新。这些结果表明 EMPO$^2$ 是构建更具探索性和泛化能力的大语言模型智能体的一个极具前景的框架。

## Abstract
Exploration remains the key bottleneck for large language model agents trained with reinforcement learning. While prior methods exploit pretrained knowledge, they fail in environments requiring the discovery of novel states. We propose Exploratory Memory-Augmented On- and Off-Policy Optimization (EMPO$^2$), a hybrid RL framework that leverages memory for exploration and combines on- and off-policy updates to make LLMs perform well with memory while also ensuring robustness without it. On ScienceWorld and WebShop, EMPO$^2$ achieves 128.6% and 11.3% improvements over GRPO, respectively. Moreover, in out-of-distribution tests, EMPO$^2$ demonstrates superior adaptability to new tasks, requiring only a few trials with memory and no parameter updates. These results highlight EMPO$^2$ as a promising framework for building more exploratory and generalizable LLM-based agents.

### 1. 核心问题与整体含义
该论文针对 **大语言模型 (LLM) 智能体在强化学习 (RL) 训练中的“探索瓶颈”** 展开研究。
*   **研究动机**：现有的 LLM 智能体在交互式环境中往往过度依赖预训练阶段的先验知识（利用），而缺乏系统性的探索能力。在需要发现新状态或尝试非直觉行为的复杂任务（如 ScienceWorld）中，传统的在线 RL 算法（如 GRPO）容易陷入局部最优，无法通过失败进行有效学习。
*   **核心目标**：建立一个统一框架，通过结合“非参数化记忆更新”和“参数化策略优化”，提升智能体的探索效率，并使其在推理阶段即使不依赖外部记忆也能保持强大的性能。

### 2. 方法论
论文提出了 **EMPO²** 框架，其核心思想是利用记忆作为探索的“脚手架”，并通过混合优化将其内化到模型参数中。

*   **自我生成记忆 (Self-Generated Memory)**：
    *   智能体在每次尝试后，利用 LLM 自身的能力对轨迹进行反思，生成自然语言形式的“提示 (Tips)”。
    *   这些 Tips 记录了失败原因或关键发现，并存储在外部记忆库中。
*   **混合采样与更新模式**：
    *   **采样阶段**：以概率 $p$ 选择“记忆增强提示”（检索相关 Tips 放入 Prompt），以 $1-p$ 选择“无记忆提示”。
    *   **更新阶段**：
        *   **同策略更新 (On-Policy)**：直接优化当前采样模式下的行为。
        *   **异策略更新 (Off-Policy)**：这是关键创新。将“带记忆引导”下产生的高奖励行为，作为目标来更新“不带记忆”的基础策略。这本质上是一种**奖励引导的知识蒸馏**，使模型学会即使没有 Tips 也能做出正确的决策。
*   **内在奖励 (Intrinsic Rewards)**：
    *   引入基于状态新颖性的奖励。通过计算当前状态与记忆中历史状态的余弦相似度，鼓励智能体访问未曾见过的状态，维持策略熵。
*   **稳定性技术**：
    *   引入掩码机制 (Masking)，抑制低概率 Token 导致的梯度爆炸，解决异策略训练中的不稳定性问题。

### 3. 实验设计
*   **实验场景**：
    1.  **ScienceWorld**：包含 19 个科学实验任务（如电路连接、化学混合），要求极强的多步规划和环境探索能力。
    2.  **WebShop**：HTML 环境下的在线购物任务，涉及搜索、导航和属性匹配。
*   **对比基准 (Baselines)**：
    *   **Naive**：原始 Qwen2.5-7B-Instruct 模型。
    *   **非参数化方法**：Reflexion（仅更新记忆，不更新参数）。
    *   **离线强化学习**：Retrospex。
    *   **在线强化学习**：GRPO（DeepSeek 提出）、GiGPO（针对智能体优化的 GRPO 变体）。

### 4. 资源与算力
*   **硬件环境**：使用了 **8 张 NVIDIA A100 GPU**（部分实验提及 40GB 版本，部分为 80GB）。
*   **基础模型**：主要基于 **Qwen2.5-7B-Instruct**。
*   **软件框架**：基于 `verl`（分布式 RL 库）和 `Agent Lightning` 框架进行开发。
*   **训练细节**：ScienceWorld 任务中，每个任务使用前 5 个变体训练，在 20 个未见变体上测试。

### 5. 实验数量与充分性
*   **实验规模**：涵盖了 ScienceWorld 的 19 个不同类别的任务和 WebShop 的标准测试集。
*   **消融实验**：非常充分。作者分别验证了：
    *   同策略与异策略组合的必要性。
    *   内在奖励对防止策略坍缩的作用。
    *   超参数 $p$（记忆采样概率）和 $q$（异策略更新概率）的敏感度分析。
*   **客观性**：通过 ID（分布内）和 OOD（分布外）双重测试验证了模型的泛化能力。在 OOD 测试中，展示了模型在完全陌生的任务上通过少量记忆尝试即可快速适应的能力。

### 6. 主要结论与发现
*   **性能飞跃**：在 ScienceWorld 上，EMPO² 比 GRPO 提升了 **128.6%**；在 WebShop 上提升了 **11.3%**。
*   **知识内化**：证明了通过异策略蒸馏，模型可以将外部记忆中的探索经验转化为自身的参数权重。
*   **泛化与适应**：EMPO² 训练出的模型不仅在原始任务上更强，还获得了一种“利用记忆进行探索”的元能力，在面对全新领域的任务时，其 Few-shot 适应速度远超基准模型。

### 7. 优点
*   **混合优化的巧妙性**：解决了“依赖记忆导致模型本身不强”和“不使用记忆导致探索效率低”的矛盾。
*   **探索机制的完备性**：结合了高层的语义反思（Tips）和底层的数值奖励（内在奖励），多维度引导探索。
*   **实用性强**：推理时不需要复杂的 Tips 检索也能获得高性能，降低了部署成本。

### 8. 不足与局限
*   **计算开销**：训练期间生成 Tips 和检索记忆增加了约 **19% 的额外耗时**。
*   **检索机制尚浅**：目前仅采用简单的向量相似度检索，对于极其复杂的长程记忆管理可能存在瓶颈。
*   **模型规模覆盖**：实验主要集中在 7B 规模的模型，尚未验证该方法在超大规模模型（如 70B 或 400B+）上的边际收益。
*   **任务领域**：虽然在科学实验和购物任务上表现优异，但在代码生成、数学证明等逻辑严密性极高且反馈稀疏的领域仍需进一步验证。
