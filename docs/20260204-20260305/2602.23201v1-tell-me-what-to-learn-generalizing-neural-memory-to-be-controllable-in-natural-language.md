## 摘要
现代机器学习模型被部署在多样化且非平稳的环境中，必须不断适应新任务和不断演进的知识。持续微调和上下文学习成本高昂且脆弱，而神经记忆方法则有望实现轻量级更新并最大限度地减少遗忘。然而，现有的神经记忆模型通常假设单一的固定目标和同质信息流，导致用户无法控制模型随时间推移记住或忽略的内容。为了应对这一挑战，我们提出了一种通用的神经记忆系统，该系统能够根据自然语言指定的学习指令执行灵活的更新。我们的方法使自适应智能体能够从异构信息源中进行选择性学习，从而支持医疗保健和客户服务等固定目标记忆更新不足的应用场景。

## Abstract
Modern machine learning models are deployed in diverse, non-stationary environments where they must continually adapt to new tasks and evolving knowledge. Continual fine-tuning and in-context learning are costly and brittle, whereas neural memory methods promise lightweight updates with minimal forgetting. However, existing neural memory models typically assume a single fixed objective and homogeneous information streams, leaving users with no control over what the model remembers or ignores over time. To address this challenge, we propose a generalized neural memory system that performs flexible updates based on learning instructions specified in natural language. Our approach enables adaptive agents to learn selectively from heterogeneous information sources, supporting settings, such as healthcare and customer service, where fixed-objective memory updates are insufficient.

### 2. 方法论
*   **核心思想**：将记忆管理建模为一个显式的指令条件化过程。记忆更新不再是无差别的吸收，而是根据指令 $I_t$ 对文档 $D_t$ 进行选择性压缩。
*   **关键技术细节**：
    *   **更新规则**：定义为 $M_t = U_\psi(M_{t-1}, I_t, D_t)$。其中 $M$ 是记忆状态，$I$ 是自然语言指令，$D$ 是新文档。
    *   **架构实现**：基于 **MemoryLLM**（以 Llama-3 为骨干）。记忆被实例化为预置在 Transformer 每一层前的连续嵌入（Prefix Embeddings）。每次更新会产生 256 个新记忆 token，随机覆盖记忆库中原有的部分（总容量约 7k tokens）。
    *   **训练目标**：通过最小化序列损失 $L_{seq}$ 进行训练。模型不仅要学会根据指令提取事实，还要学会忽略干扰信息。
*   **算法流程**：在训练阶段，模型接收“文档+指令”对，然后通过一系列“探测查询（Probes）”来验证其是否正确更新了记忆（包括正面事实、负面干扰和拒绝行为）。

### 3. 实验设计
*   **数据集/场景**：构建了一个基于 **CounterFACT** 的合成基准测试。

    *   **OOD（分布外）测试**：专门预留了 4 个类别和相关指令不参与训练，以测试模型对**全新指令**的泛化能力。
*   **Benchmark 任务**：
    1.  **事实学习**：选择性记忆特定类别的事实。
    2.  **样式/格式学习**：学习文档的 Markdown 格式（如 JSON, XML）而忽略内容。
    3.  **拒绝行为**：根据指令拒绝回答特定类别的敏感信息。
    4.  **组合泛化**：同时处理多种指令（如“学习 A 类事实并拒绝 B 类事实”）。
*   **对比方法**：
    *   **MemoryLLM**：原始的无指令神经记忆模型。
    *   **ICL-FT**：经过微调的上下文学习（将所有历史文档和指令放入 Prompt）。
    *   **RAG-FT**：经过微调的检索增强生成。

*   **硬件设备**：使用了 **6 台 NVIDIA A100 GPU**。
*   **训练细节**：采用梯度累加，有效 Batch Size 为 6 个 episodes（共 24 个文档）。使用 8-bit AdamW 优化器。
*   **训练时长**：采用早停法（Early Stopping），所有模型均在 10 个 Epoch 内收敛。

### 5. 实验数量与充分性
*   **实验规模**：训练集包含约 1.1 万个文档，测试集包含 2,276 个 OOD 文档。
*   **充分性分析**：
    *   **多维度评估**：不仅测试了准确率，还测试了**特异性**（是否误改了邻近事实）和**选择性**（是否成功忽略了干扰项）。

### 6. 主要结论与发现
*   **泛化能力强**：GNM 能够完美理解并执行在训练中从未见过的自然语言指令。
*   **选择性优异**：在“忽略干扰信息”方面，GNM 显著优于 ICL 和 RAG。ICL 即使经过微调，也容易受到上下文干扰信息的影响。
*   **计算效率高**：GNM 的推理成本是 $O(1)$，不随学习文档数量增加而增长，而 ICL 的成本随历史线性/二次增长。
*   **内部机制**：研究发现模型的前几层（5-14层）负责编码指令，中后层利用这些指令信息来决定哪些数据该写入记忆。

### 7. 优点
*   **创新性**：首次将神经记忆从“固定目标”泛化为“语言可控”，填补了持续学习领域的一个重要空白。
*   **实用性**：解决了 RAG 和 ICL 在处理异构、冲突信息流时的控制力不足问题。
*   **高效性**：在保持高性能的同时，极大地降低了长程依赖的计算开销。

### 8. 不足与局限
*   **数据局限**：目前仅在合成基准（CounterFACT）上验证，尚未在真实世界的大规模非结构化数据集上测试。
*   **记忆容量限制**：由于底层 MemoryLLM 的限制，当更新步数超过 20 步时，旧记忆会被覆盖，导致性能下降。
*   **冲突处理**：模型目前缺乏处理时间顺序或矛盾信息的显式机制（例如，如果两条指令相互矛盾，模型可能无法按时间先后排序）。
*   **应用限制**：目前主要针对文本事实，对于更复杂的逻辑推理或多模态信息的选择性记忆尚待探索。
